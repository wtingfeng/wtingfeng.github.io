15、回文链表

2020-07-14

题目描述：请判断一个链表是否为回文链表。
如 输入：1->2 输出：false
输入：1->2->2->1 输出：true

思路分析：将链表的后半部分反转，然后将前半部分和后半部分进行比较。比较完成后将链表恢复原样。
class Solution{
public:
  bool isPalindrome(ListNode *head){
    if(!head || !head->next)
      return true;
    ListNode *fast=head, *slow=head, *pre=nullptr;
    while(fast&&fast->next){
      slow=slow->next;
      fast=fast->next->next;
    }
    while(slow){
      ListNode *p=slow->next;
      slow->next=pre;
      pre=slow;
      slow=p;
    }
    while(head&&pre){
      if(head->val!=pre->val)
        return false;
      head=head->next;
      pre=pre->next;
    }
    return true;
  }
}
