8、删除排序链表中的重复元素

2020-07-03

题目描述：给定一个排序链表，删除所有重复的元素，使得每个元素只出现一次。

思路：方法一、由于输入的链表是有序的，只需要通过比较当前节点的值与其后继结点的值，若相等则表明为后继结点为重复结点，更改当前结点的next指针，使其指向重复结点的下一个结点。

struct ListNode {
  int val;
  ListNode *next;
  ListNode(int x) : val(x), next(NULL) {}
};
class Solution {
public:
  ListNode * deleteDuplicates(ListNode *head) {
    ListNode *current=head;
    while(current!=nullptr &&current->next!=nullptr){
      if(current->valcurrent->next->val)
        current->next=current->next->next;
      else
        current=current->next;
    }
    return head;
  }
};
方法二、使用递归解决该问题，递归的终止条件为当head指向的链表只有一个元素时，该链表无重复元素，此时即可返回，返回的应该是链表的头结点。若head->next指向了一个去重链表，此时判断head->val和head->next->val是否相等，若相等，则说明元素重复，直接返回head->next,若不等则无重复元素，直接返回head。
struct ListNode {
  int val;
  ListNode *next;
  ListNode(int x) : val(x), next(NULL) {}
};
class Solution {
public:
  ListNode * deleteDuplicates(ListNode *head) {
    if(!head || !head->next)
      return head;
    ListNode *next = head->next;
    head->next = deleteDuplicates(next);
    if(head->valnext->val)
      head=head->next;
    return head;
  }
};
