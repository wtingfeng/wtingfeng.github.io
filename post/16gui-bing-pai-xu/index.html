16、归并排序

2020-07-28
归并排序是典型的分治思想的代表，首先把原问题分解为两个或多个子问题，然后求解子问题的解，最后使用子问题的解来构造出原问题的解。
给定一个待排序的数组，首先把该数组划分为两个子数组，然后对子数组进行排序（递归调用归并排序），最后对两个有序的子数组进行合并，使合并之后的数组为有序状态。
1、归并排序采用了分治思想，它的核心在于合并子问题的解而不是求解子问题。
2、归并排序不是原址排序，它有排序过程中需要借助额外的内存空间。
3、归并排序为稳定排序
4、归并排序的时间复杂度为O(NlogN).

一、基于递归的实现方法
分解：将当前待排序的序列A[low],...,A[high]一分为二，分裂点mid=(low+high)/2
求解：递归地对序列A[low],...,A[mid]和A[mid+1],...,A[high]进行归并排序
组合：将两个已排序子序列归并为一个有序序列
递归的终止条件：子序列长度为1
void MergeSort(int arr[], int arr1[],int low, int high){
  if(low>=high)
    return;
  int mid=low+(high-low)/2;
  MergeSort(arr, arr1, low, mid);
  MergeSort(arr, arr1, mid+1, high);
  merge(arr1, arr, low, mid, high);
}
void merge(int arr[], int arr1[],int low, int mid, int high){
  int i=low,j=mid+1,k=0;
  while(i<=mid&&j<=high){
    if(arr[i]<=arr[j])
      arr1[k++]=arr[i++];
    else
      arr1[k++]=arr[j++];
  }
  while(i<=mid)
    arr1[k++]=arr[i++];
  while(j<=high)
    arr1[k++]=arr[j++];
  for(i=low,k=0;i<=high;i++,k++)
    arr[i]=arr1[k];
}
