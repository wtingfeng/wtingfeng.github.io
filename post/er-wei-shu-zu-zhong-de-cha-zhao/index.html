12、二维数组中的查找

2020-07-12

题目描述：在一个二维数组中（每个一维数组的长度相同），每一行都按照从左到右递增的顺序排序，每一列都按照从上到下递增的顺序排序。请完成一个函数，输入这样的一个二维数组和一个整数，判断数组中是否含有该整数。

思路分析：把二分值定在右上角或者左下角，就可以进行二分。若以右上角为例，设初始值为右上角元素，如果val==target，直接返回；如果target>val，说明target在更大的位置，val左边的元素都小于val，也就是小于target，说明本行无效，则将val下移到下一行；如果target<val，说明target在更小的位置，val下边的元素都是都是大于val，也就是大于target，说明本列下方无效，将val左移到前一列。
class Solution {
public:
  bool Find(int target, vector<vector < int > > array) {
    int rol=array.size();
    int col=array[0].size();
    int i=0,j=col-1;
    while(i<=rol-1 && j>=0){
      if(array[i][j]==target)
        return true;
      else if(array[i][j]>target)
        j--;
        else
          i++;
    }
    return false;
  }
};
