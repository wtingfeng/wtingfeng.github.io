17、从前序和中序遍历序列构造二叉树

2020-07-29

题目描述：根据一棵树的前序遍历和中序遍历构造二叉树。

方法一、递归。 时间复杂度O(n) 空间复杂度O(n)
对于任意一棵树而言，前序遍历的形式为：[根节点, [左子树前序遍历结果], [右子树前序遍历结果]]，根节点总是前序遍历的第一个节点。而中序遍历的形式总是[[左子树中序遍历结果], 根节点, [右子树中序遍历结果]]，只要在中序遍历中定位到根节点，就可以分别知道左子树和右子树中的结点数目。对应到前序遍历的结果中，我们就知道了左子树的前序、中序遍历结果，以及右子树的前序、中序遍历结果。我们就可以递归地构造左子树和右子树，再将两棵子树接到根节点的左右位置。
struct TreeNode{
  int val;
  TreeNode * left;
  TreeNode * right;
  TreeNode(int x):val(x),left(nullptr),right(nullptr){}
};
//unordered_map< int,int > index;
TreeNode* buildTree(vector< int >& preorder, vector< int >& inorder){
  /* 可以考虑使用hashmap帮助快速定位根节点
  for(int i=0;i<preorder.size();i++)
    index[inorder[i]]=i; */
  return Struct(0, preorder.size()-1, 0, inorder.size()-1, preorder, inorder);
}
TreeNode Struct(int leftPre, int rightPre, int leftIn, int rightIn, vector< int >& preorder, vector< int >& inorder){
  if(leftPre>rightPre || leftIn>rightIn)
    return nullptr;
  TreeNode *root=new TreeNode(preorder[leftPre]);
  int start=leftIn;
  while(start<=rightIn && inorder[start]!=preorder[leftPre])
    start++;
  //start=index[preorder[leftPre]];
  root->left=Struct(leftPre+1,leftPre+start-leftIn,leftIn,start-1,preorder,inorder);
  root->right=Struct(leftPre+start-leftIn+1,rightPre,start+1,rightIn,preorder,inorder);
  return root;
}
