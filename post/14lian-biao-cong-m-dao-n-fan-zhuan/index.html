14、链表从m到n反转

2020-07-14

题目描述：将一个链表从m位置反向到n，并在一个通道内完成。例如:
给定1 - > 2 - > 3 - > 4 - > 5 - > NULL,m = 2和n = 4
返回1 - > 4 - > 3 - > 2 - > 5 - > NULL
注意:给定m, n满足以下条件:1 ≤ m ≤ n ≤ 链表长度

class Solution {
public:
  ListNode* reverseBetween(ListNode* head, int m, int n) {
    ListNode *root = new ListNode(0);
    root->next=head;
    ListNode *p=root,*q=root;
    m--;
    while(m--) p=p->next;
    while(n--) q=q->next;
    ListNode *tail = q->next;
    q->next=nullptr;
    ListNode *curr = p->next,*pro = tail;
    while(curr){
      ListNode *temp = curr->next;
      curr->next=pro;
      pro=curr;
      curr=temp;
    }
    p->next=pro;
    return root->next;
  }
};
