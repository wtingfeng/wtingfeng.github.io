10、反转链表

2020-07-03

题目描述：定义一个函数，输入一个链表的头节点，反转该链表并输出反转后链表的头节点。

思路：方法一、双指针迭代。
1、定义两个指针，pre和cur，pre在前，cur在后。
2、每次让cur的next指向pre，实现一次局部反转。
3、局部反转完成后，pre和cur同时往前移动一个位置。
4、循环上述过程，直到cur到达链表尾部。

class Solution {
public:
  ListNode * reverseList(ListNode *head) {
    ListNode *pre=nullptr, *cur=head;
    while(cur){
      ListNode *tmp=cur->next;
      cur->next=pre;
      pre=cur;
      cur=tmp;
    }
    return pre;
  }
};

方法二、递归。
递归的终止条件：当前结点为null，或者下一个结点为null。
1、使用递归函数，一直递归到链表的最后一个结点，该结点就是反转后的头结点。
2、每次函数在返回的过程中，让当前结点的下一个结点的next指向当前结点。
3、同时让当前结点的next指向null，从而实现从链表尾部开始的局部反转。
4、当递归函数全部出栈后，链表反转完成。
class Solution {
public:
  ListNode * reverseList(ListNode *head) {
    if(head == nullptr || head->next == nullptr)
      return head;
    ListNode *ret=reverseList(head->next);
    head->next->next=head;
    head->next=nullptr;
    return ret;
  }
};
